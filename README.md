\# Online Sales Analysis



Python OOP project for managing products, inventory and shopping cart.



\## Classes



\### Product

Contains product name, price and quantity.



\### ProductManager

Manages available products. It can add, display and remove products.



\### Cart

Manages shopping cart items. It can add products, display cart content and calculate total value.
